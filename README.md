# 🚦 Traffic Accident Analytics Dashboard

## 📌 Project Overview

This project analyzes traffic accident data to identify trends, accident severity, weather impacts, road conditions, and accident hotspots. The dashboard provides actionable insights using Python, SQL, Excel, and Power BI.

---

## 🛠 Tools Used

- Python (Pandas, NumPy)
- MySQL
- Microsoft Excel
- Power BI

---

## 📊 Dashboard Features

- Total Accidents
- Total Casualties
- Average Risk Score
- Average Vehicles Involved
- Monthly Accident Trend
- Top 5 Cities by Accidents
- Accident Severity Analysis
- Weather Condition Analysis
- Road Type Analysis
- Day-wise Accident Analysis

---

## 📂 Project Workflow

Raw Dataset → Excel → Python (Data Cleaning & EDA) → MySQL → Power BI Dashboard
